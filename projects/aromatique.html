Aromatique

Web-Based Point of Sale System

Screenshots

Project Overview

Aromatique is a web-based Point of Sale (PoS) and inventory management system developed using Node.js during my third year at university. As a team project, it provided hands-on experience in backend web development, database integration with PostgreSQL, and collaborative software development while building a complete business management application.

My Contributions

  • Developed REST API endpoints for inventory and sales management using Node.js.
  • Designed and integrated the PostgreSQL database schema and backend data operations.
  • Implemented inventory tracking and sales transaction management features.
  • Integrated frontend components with backend APIs to enable seamless data flow.
  • Coordinated module integration, resolved compatibility issues, and participated in system testing.

Technical Implementation

Aromatique uses a Node.js backend with RESTful APIs and PostgreSQL for data persistence. The application follows a modular structure to manage products, inventory, and sales while enabling communication between the frontend and backend.

Technologies Used

  • Node.js
  • PostgreSQL
  • JavaScript (Frontend)
  • RESTful API

Key Features

  • Product and inventory management
  • Sales transaction processing
  • RESTful API backend
  • PostgreSQL database integration
  • Role-based user access
